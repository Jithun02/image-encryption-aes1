Cryptographic Architecture

AES-256-CBC Encryption

Encrypts image bits in 128-bit blocks using the Advanced Encryption Standard (AES) with a 256-bit key length, the global standard for securing sensitive government data.

PBKDF2 Key Derivation

Passwords are never used directly. We pass them through a Password-Based Key Derivation Function 2 (PBKDF2) for 1000 iterations to harden against dictionary attacks.

Hex Salt & Initialization Vector

Every file generates a unique, pseudo-random salt and IV. This prevents duplicate file inputs from yielding identical ciphertexts (avoiding statistical scanning patterns).

Browser Sandbox Security

There are no servers or APIs. All computation is handled inside the safe local sandbox of your browser. Disconnecting the internet will not halt the application.
